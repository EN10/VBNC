Visual Basic .Net Online
=

1. Register on www.edx.org  
2. Login to www.cs50.io 

Open Terminal:  
Click "+" -> New Teminal OR Alt + T     

You can use Install, Compile & Run commands     

Install (Only Once):
-
    sudo apt update     
    sudo apt install mono-vbnc      

Compile (Make .exe):    
-
    vbnc Module1.vb 

Run:    
-
    mono Module1.exe    

Advanced
-
Setup:  
    chmod 777 run.sh    
    
Compile & Run:   
    ./run.sh
